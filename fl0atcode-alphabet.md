# fl0atcode
remaking the alphabet to make it look like spammed text™

chars used:
```
% / < > : ! - + = #
```

---

## alphabet

### vowels (easy + common)
> [!TIP]
> vowels always start with "%"!

| letter | yerocode |
|-|-|
| a | %< |
| e | %> |
| i | %! |
| o | %: |
| u | %+ |
| y | %= |
---

### consonants
> [!TIP]
> consonants always start with "/"

| letter | yerocode |
|-|-|
| b | /< |
| c | /> |
| d | /: |
| f | /! |
| g | /- |
| h | /<< |
| j | /:+ |
| k | /+> |
| l | /<> |
| m | /% |
| n | /%> |
| p | /<+ |
| q | /== |
| r | /:> |
| s | /%= |
| t | /!> |
| v | /+< |
| w | /%% |
| x | /-> |
| z | /:= |

## examples

**english:**
```
fl0atpoint
```

